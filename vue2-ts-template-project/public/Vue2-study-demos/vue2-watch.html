Vue2 Watch 监听器详解

深入理解 Vue2 的 watch 功能及各种使用技巧

基本监听

当前值: {{ counter }}
变化历史:

{{ log }}

// 监听简单值的变化 watch: { counter(newVal, oldVal) { this.basicLogs.unshift(`计数器从 ${oldVal} 变为 ${newVal}`); } }

对象深度监听

用户信息: {{ user }}
变化历史:

{{ log }}

// 深度监听对象 watch: { user: { handler(newVal, oldVal) { this.objectLogs.unshift('用户对象发生变化'); }, deep: true, // 深度监听 immediate: true // 立即触发一次 } }

监听特定属性

当前产品: {{ selectedProduct.name }} - ${{ selectedProduct.price }}
变化历史:

{{ log }}

// 监听对象的特定属性 watch: { 'selectedProduct.price': function(newPrice, oldPrice) { this.propertyLogs.unshift(`价格从 $${oldPrice} 变为 $${newPrice}`); } }

监听计算属性

原始文本: "{{ text }}"
反转文本: "{{ reversedText }}"
变化历史:

{{ log }}

// 计算属性 computed: { reversedText() { return this.text.split('').reverse().join(''); } }, // 监听计算属性 watch: { reversedText(newVal) { this.computedLogs.unshift(`反转文本变为: "${newVal}"`); } }

多数据源监听

当前面积: {{ area }}
变化历史:

{{ log }}

// 使用数组监听多个数据源 watch: { // 监听多个数据源 sources: { handler([width, height]) { this.multiLogs.unshift(`尺寸变化: ${width} x ${height}`); }, deep: true } }, computed: { area() { return this.width * this.height; }, sources() { return [this.width, this.height]; } }

异步操作与防抖

搜索结果: {{ searchResults }}
变化历史:

{{ log }}

// 异步操作与防抖 watch: { searchQuery: { handler: _.debounce(function(newVal) { if (newVal) { this.asyncLogs.unshift(`正在搜索: "${newVal}"`); // 模拟API请求 setTimeout(() => { this.searchResults = `找到 ${Math.floor(Math.random() * 100)} 条 "${newVal}" 的结果`; }, 1000); } else { this.searchResults = ''; } }, 500) // 500ms防抖 } }
