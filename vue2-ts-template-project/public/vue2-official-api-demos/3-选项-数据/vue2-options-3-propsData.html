Vue2 propsData 选项示例

介绍：

propsData 主要用于通过 Vue.extend 创建组件构造器后，new 组件实例时传递 props 数据。
常用于手动挂载组件或单元测试场景。
注意： 只有通过 JavaScript 代码 new 的方式才用得到 propsData，模板中用不到。 

// 基本用法
const MyComp = Vue.extend({
  props: ['msg'],
  template: '<div>消息：<b>{{ msg }}</b></div>'
});
const vm = new MyComp({
  propsData: {
    msg: 'Hello from propsData!'
  }
});
vm.$mount('#mount-point');

示例：propsData 传递 props


const MyComp = Vue.extend({
  props: ['msg'],
  template: '<div>消息：<b>{{ msg }}</b></div>'
});
const vm = new MyComp({
  propsData: {
    msg: 'Hello from propsData!'
  }

vm.$mount('#mount-point');

propsData 说明

说明：

propsData 只在通过 new 组件构造器 的方式创建实例时有效。
在模板语法（<my-comp :msg="xxx">）中不会用到 propsData。
一般用于手动挂载、动态创建组件、单元测试等场景。
