Vue.use 插件注册示例

介绍：

Vue.use 用于安装 Vue 插件。插件通常用于为 Vue 添加全局功能，比如全局方法、指令、过滤器等。
语法： Vue.use(插件[, 可选参数]) 

// 定义一个简单插件
const MyPlugin = {
  install(Vue, options) {
    // 添加全局方法
    Vue.prototype.$hello = function (name) {
      alert('你好, ' + (name || '游客') + '！');
    }
  }
}

// 安装插件
Vue.use(MyPlugin);

示例：自定义插件添加全局方法


const MyPlugin = {
  install(Vue, options) {
    Vue.prototype.$hello = function (name) {
      alert('你好, ' + (name || '游客') + '！');
    }
  }
}
Vue.use(MyPlugin);

// 使用：this.$hello('张三')

插件还能做什么？

说明：

插件可以扩展 Vue 的功能，比如：
- 添加全局方法或属性
- 添加全局指令
- 添加全局过滤器
- 注入组件选项
- 添加 Vue 实例方法等
