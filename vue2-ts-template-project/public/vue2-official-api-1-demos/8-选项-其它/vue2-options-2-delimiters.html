Vue2 选项 delimiters 示例

介绍：

delimiters 选项用于自定义文本插值的分隔符（默认是 {{ }}）。
主要用于与其它模板引擎（如后端的 Jinja2、Django、Blade 等）冲突时，避免语法冲突。
仅在使用 new Vue({ delimiters: ... }) 方式创建实例时有效。

示例1：默认分隔符 {{ }}

默认插值：{{ message }} 
// 默认分隔符
new Vue({
  el: '#default-delimiters',
  data: {
    message: 'Hello, 默认分隔符！'
  }
});
说明：

默认情况下，插值分隔符为 {{ }}。

示例2：自定义分隔符 [[ ]]

自定义插值源码是这样的： [ [ message ] ]

自定义插值，值为：[[ message ]]

// 自定义分隔符
new Vue({
  el: '#custom-delimiters',
  data: {
    message: 'Hello, 自定义分隔符！'
  },
  delimiters: ['[[', ']]']
});
说明：

当和后端模板语法冲突时，可以通过 delimiters 选项自定义插值分隔符。

示例3：和后端模板引擎冲突时的应用场景

场景说明：

如后端模板引擎（如 Jinja2）也使用 {{ }}，此时前端 Vue 可用 [[ ]]，后端继续用 {{ }}，互不干扰。



后端变量：{{ backend_var }}
前端变量：[[ frontend_var ]]
