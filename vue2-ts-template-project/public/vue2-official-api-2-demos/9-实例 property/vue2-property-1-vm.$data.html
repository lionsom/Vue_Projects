Vue2 实例属性 vm.$data 示例

介绍：

vm.$data 是 Vue 实例的一个属性，指向用于数据响应的对象。
你可以通过 this.$data 访问到 data 选项中的所有数据。
通常不建议直接替换 vm.$data，但可以用于调试或查看数据结构。 

// 访问实例数据对象
console.log(vm.$data);

示例1：展示 vm.$data 内容

姓名：{{ name }}
年龄：{{ age }}
// data 选项
data: {
  name: '小明',
  age: 18
}
// 访问实例数据对象
console.log(this.$data);

示例2：打印 vm.$data 到控制台

// 打印 this.$data
this.$data
// 也可以直接通过 vm.$data 访问

更多说明

- vm.$data 始终指向 data 选项的对象。
- 不建议直接替换整个 vm.$data，但可以读取和调试。
- this.$data === this 返回 false，但 this.$data.name === this.name 返回 true。
