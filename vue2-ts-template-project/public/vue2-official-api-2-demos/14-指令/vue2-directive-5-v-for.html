Vue2 内置指令 v-for 示例

介绍：

v-for 指令用于基于一个数组或对象来渲染一个列表。
语法：v-for="item in items"，可选带索引：v-for="(item, index) in items"。
建议为每个列表项绑定唯一的 key，以提升渲染性能和避免渲染错误。 

<li v-for="item in items" :key="item.id">{{ item.text }}</li>

示例1：遍历数组

  • {{ fruit }}
<li v-for="fruit in fruits" :key="fruit">{{ fruit }}</li>
<input v-model="newFruit">
<button @click="addFruit">添加</button>

示例2：遍历对象

  • {{ key }}: {{ value }} 
<li v-for="(value, key) in user" :key="key">
  {{ key }}: {{ value }}
</li>

示例3：遍历数组并获取索引

  • {{ idx }} - {{ lang }} 
<li v-for="(lang, idx) in languages" :key="lang">
  {{ idx }} - {{ lang }}
</li>

示例4：遍历对象并获取索引

  • {{ idx }} - {{ key }}: {{ value }} 
<li v-for="(value, key, idx) in user" :key="key">
  {{ idx }} - {{ key }}: {{ value }}
</li>

示例5：遍历数组对象并使用 key

编号 姓名 年龄
{{ person.id }} {{ person.name }} {{ person.age }} 
<tr v-for="person in people" :key="person.id">
  <td>{{ person.id }}</td>
  <td>{{ person.name }}</td>
  <td>{{ person.age }}</td>
</tr>

更多说明

- v-for 支持遍历数组、对象、字符串、数字、甚至是自定义迭代器。
- 建议为每个列表项绑定唯一的 key，通常用 id 或唯一字段。
- 遍历对象时，参数顺序为 (value, key, index)。
- 参考：Vue2 官方文档 v-for
