Vue2 内置指令 v-text 示例

介绍：

v-text 指令用于更新元素的 textContent。
它的作用类似于 {{ }} 插值表达式，但不会解析 HTML，仅渲染纯文本。
使用 v-text 可以避免 XSS 攻击风险，适合需要动态插入纯文本的场景。 

<span v-text="msg"></span>
等价于
<span>{{ msg }}</span>

示例1：基本用法

<span v-text="message"></span>
<input v-model="message">

示例2：与插值表达式和 v-html 的区别

（v-text 渲染）
{{ htmlStr }} （插值渲染）
（v-html 渲染） 
<span v-text="htmlStr"></span>
<span>{{ htmlStr }}</span>
<span v-html="htmlStr"></span>

注意：v-text 和 {{}} 都不会解析 HTML 标签，v-html 会解析并渲染 HTML。

更多说明

- v-text 适合动态插入纯文本内容，防止 XSS 攻击。
- v-text 会覆盖元素内原有的内容。
- v-text 不能和插值表达式 {{}} 混用在同一个元素上。
