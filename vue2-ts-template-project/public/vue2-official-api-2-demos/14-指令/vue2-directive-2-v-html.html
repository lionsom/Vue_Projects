Vue2 内置指令 v-html 示例

介绍：

v-html 指令用于更新元素的 innerHTML，可以将一段 HTML 字符串渲染为真正的 HTML 结构。
与 {{ }} 或 v-text 不同，v-html 会解析并渲染 HTML 标签。
注意：使用 v-html 可能导致 XSS 攻击风险，务必确保内容安全！ 

<div v-html="htmlContent"></div>
等价于
<div>{{ htmlContent }}</div> // 但不会解析HTML

示例1：基本用法

<div v-html="htmlContent"></div>
<input v-model="htmlContent">

示例2：与插值表达式和 v-text 的区别

（v-html 渲染）
（v-text 渲染）
{{ htmlStr }} （插值渲染） 
<div v-html="htmlStr"></div>
<span v-text="htmlStr"></span>
<span>{{ htmlStr }}</span>

注意：只有 v-html 会解析并渲染 HTML 标签，v-text 和 {{}} 都只渲染纯文本。

更多说明

- v-html 适合需要动态渲染 HTML 结构的场景。
- 使用 v-html 时要特别注意内容的安全性，防止 XSS 攻击。
- v-html 会覆盖元素内原有的内容。
- 参考：Vue2 官方文档 v-html
