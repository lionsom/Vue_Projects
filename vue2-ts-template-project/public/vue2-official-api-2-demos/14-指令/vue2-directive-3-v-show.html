Vue2 内置指令 v-show 示例

介绍：

v-show 指令用于根据表达式的真假值，切换元素的可见性。
本质上是通过 display: none 控制元素的显示与隐藏。
与 v-if 不同，v-show 不会销毁或重建 DOM 元素，只是简单地切换 CSS。
适合频繁切换显示/隐藏的场景。 

<div v-show="isVisible">内容</div>
等价于
<div style="display: none;">内容</div> // 当 isVisible 为 false 时

示例1：基本用法

这段内容会根据 isVisible 的值显示或隐藏 
<button @click="isVisible = !isVisible">切换显示/隐藏</button>
<div v-show="isVisible">这段内容会根据 isVisible 的值显示或隐藏</div>

示例2：与 v-if 的区别

v-show：我只是隐藏/显示
v-if：我会被销毁和重建
<div v-show="show">v-show：我只是隐藏/显示</div>
<div v-if="show">v-if：我会被销毁和重建</div>

注意：v-show 只是切换 display，v-if 是条件渲染，频繁切换建议用 v-show。

更多说明

- v-show 不支持 <template> 元素。
- v-show 的元素在初始渲染时会被插入，无论条件真假。
- 适合需要频繁切换显示状态的场景。
